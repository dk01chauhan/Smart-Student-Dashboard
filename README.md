# Smart-Student-Dashboard
The Smart Student Dashboard is a full-stack web application designed to manage student-related activities efficiently. It provides an interactive platform for students and faculty to access academic information, manage records, and perform essential operations like registration, attendance, and performance tracking.

This project is developed using HTML, CSS, JavaScript (Frontend), Java (Backend), and SQL (Database), ensuring a complete real-world application structure.

 Features
 Student Registration & Login System
 Faculty Panel Access
 Dashboard with Student Data
 Attendance Management System
 Performance Tracking
 Authentication & Access Control
 User-friendly and responsive UI
 Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Java (Servlets / JSP or Core Java)
Database: MySQL / SQL
Tools: VS Code, Git, GitHub
